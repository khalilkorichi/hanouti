# Phase05 – سجل المبيعات والتفاصيل

## الأهداف
- فلاتر: بحث، تاريخ (اليوم/الأسبوع/الشهر/مخصص)، طريقة دفع، حالة.
- KPIs للفترة.
- جدول مبيعات + إجراءات: عرض/طباعة/إلغاء.
- نافذة تفاصيل بعناصر الفاتورة + أزرار الطباعة/الإغلاق.

## المهام
1) `GET /sales?query=&status=&pay=&from=&to=&page=&limit=`
2) KPIs Endpoint للفترة.
3) UI: شريط أدوات + DataGrid + Badges ملوّنة + إجراءات.
4) Modal تفاصيل موحّد التصميم.

## القبول
- الأداء مستقر مع آلاف الفواتير (Pagination + فهارس).
- إلغاء فاتورة بعد صلاحية محددة يتطلب إذن/سبب + Stock movement عكسي.

## اختبارات
- تكامل: فلترة/ترقيم/طباعة.
- وحدة: حاسبة KPIs للفترة.
