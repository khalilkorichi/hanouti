## 1. تفاصيل التنفيذ

### Backend Endpoints:
1. `GET /reports/kpis`: إجمالي المبيعات، الطلبات، الأرباح
2. `GET /reports/sales-over-time`: بيانات للرسم البياني الخطي
3. `GET /reports/top-products`: بيانات للرسم البياني الشريطي
4. `GET /reports/stock-status`: بيانات للرسم البياني الدائري
5. `GET /reports/profit-margin`: تحليل هامش الربح

### Frontend Components:
- **KPI Cards**: تعرض القيمة، العنوان، أيقونة، ونسبة التغير (اختياري)
- **Charts**: رسوم بيانية تفاعلية مع Tooltips وتنسيق RTL
- **Date Filter**: قائمة منسدلة لتغيير نطاق البيانات
