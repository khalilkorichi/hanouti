# Phase06 – المخزون وحركة الأصناف

## الأهداف
- جدول مخزون قوي + مؤشرات لونية (🟢🟡🔴).
- تعديل سريع (Modal/Inline) للكمية والأسعار والحدود.
- سجل حركة المخزون (stock_movements).
- تنبيهات المنتجات المنخفضة (Notifications).

## المهام
1) `GET /inventory`, `GET /stock/movements`.
2) Quick Edit يحفظ فوريًا مع Toast.
3) توليد حركة مخزون عند التعديل اليدوي (reason=adjustment).
4) إشعار تلقائي عند تجاوز حد الانخفاض.

## القبول
- كل تغيير كمية يولّد حركة حركة مسجّلة.
- المؤشرات اللونية دقيقة حسب القيم.

## اختبارات
- وحدة: قواعد المؤشرات.
- تكامل: تعديل كمية → حركة + إشعار.
