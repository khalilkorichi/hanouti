# 🚀 دليل البدء السريع - نظام حانوتي

## المتطلبات الأساسية

- Python 3.9+ مع pip
- Node.js 18+ مع npm
- محرر نصوص (VS Code مُفضّل)

---

## 🏃 التشغيل السريع

### 1️⃣ تشغيل Backend (خادم API)

```bash
cd backend
.\venv\Scripts\Activate
python -m uvicorn main:app --reload
```

✅ **Backend جاهز على**: http://localhost:8000
📚 **API Docs**: http://localhost:8000/docs

### 2️⃣ تشغيل Frontend (واجهة المستخدم)

افتح نافذة طرفية جديدة:

```bash
cd frontend
npm run dev
```

✅ **Frontend جاهز على**: http://localhost:5173

---

## 🔐 تسجيل الدخول

1. افتح المتصفح على: http://localhost:5173/login
2. **اسم المستخدم**: `admin`
3. **كلمة المرور**: `123`
4. سيُطلب منك تغيير كلمة المرور عند أول دخول ✨

## 🛠️ الميزات الأساسية

### ✅ إدارة المنتجات
1. اذهب لـ `/products`
2. اضغط "إضافة منتج"
3. املأ البيانات (الاسم، الباركود، السعر، الكمية، الفئة)
4. احفظ المنتج

### ✅ إدارة الفئات
1. اذهب لـ `/categories`
2. اضغط "إضافة فئة"
3. أدخل الاسم واختر لون
4. احفظ الفئة

### ✅ تغيير كلمة المرور
1. اذهب لـ `/settings`
2. اختر تبويب "الأمان"
3. اضغط "تغيير كلمة المرور"

### ✅ تبديل الوضع الليلي/النهاري
- اضغط على أيقونة الشمس/القمر في الـ Header

## 📊 API Endpoints

زيارة http://localhost:8000/docs للحصول على وثائق تفاعلية كاملة.

### الأساسية:
- `GET /health` - فحص صحة الخادم
- `GET /` - رسالة الترحيب

### المصادقة:
- `POST /token` - تسجيل الدخول
- `GET /users/me` - بيانات المستخدم الحالي
- `POST /change-password` - تغيير كلمة المرور

### المنتجات:
- `GET /products/` - قائمة المنتجات
- `POST /products/` - إضافة منتج
- `PUT /products/{id}` - تحديث منتج
- `DELETE /products/{id}` - حذف منتج
- `GET /products/search?q=...` - بحث

### الفئات:
- `GET /categories/` - قائمة الفئات
- `POST /categories/` - إضافة فئة
- `PUT /categories/{id}` - تحديث فئة
- `DELETE /categories/{id}` - حذف فئة

## 🎯 نصائح سريعة

### للمطورين:
1. **Hot Reload**: كل من Backend و Frontend يدعم Hot Reload
2. **TypeScript**: Frontend مكتوب بـ TypeScript بالكامل
3. **Linting**: إصلاح الأخطاء تلقائياً عند الحفظ
4. **RTL**: كل الواجهات تدعم العربية (RTL)

### للمستخدمين:
1. **الألوان**: تعبر عن الحالة (أحمر=خطر، أخضر=نجاح، أزرق=رئيسي)
2. **الكمية**: تظهر بالأحمر إذا < 10
3. **العملة**: جميع الأسعار بالدينار الجزائري (دج)
4. **الإشعارات**: ستظهر في أعلى الشاشة

## 🐛 حل المشاكل الشائعة

### المشكلة: Backend لا يعمل
**الحل**:
```bash
cd backend
python -m pip install -r requirements.txt
python setup_db.py  # إعادة إنشاء قاعدة البيانات
```

### المشكلة: Frontend لا يعمل
**الحل**:
```bash
cd frontend
npm install  # إعادة تثبيت الحزم
npm run dev
```

### المشكلة: خطأ في تسجيل الدخول
**الحل**:
1. تأكد من تشغيل Backend
2. تحقق من أن البيانات صحيحة: admin / 123
3. امسح localStorage في المتصفح (F12 → Console → `localStorage.clear()`)

### المشكلة: لا تظهر البيانات
**الحل**:
1. افتح http://localhost:8000/docs
2. جرب الـ endpoints مباشرة
3. تحقق من Console في المتصفح (F12)
